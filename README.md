# macOS_Guideline

- macOS 기반 시스템에서 정보 유출 방지 및 보안 가이드라인을 정리한 저장소입니다.

- macOS의 보안 강화를 위한 점검 방법, 로그 검사 도구 등을 함께 제공합니다.


# 소개

이 저장소는 다음을 목표로 합니다:
- macOS 환경에서 발생할 수 있는 정보 유출 시나리오 분석
- AUL (Apple Unified Logging System)을 활용한 보안 점검 및 대응 절차 제공
- 활용 가능한 로그 검사 스크립트 제공



# 구성
### macOS 정보유출 가이드라인.pdf
- macOS 시스템 보안 설정, 위험 요소 분석 및 대응 방법을 정리한 공식 가이드라인 문서입니다.  

### mac_log_checker
- macOS 로그를 분석하여 의심스러운 활동을 찾아주는 스크립트입니다.

# 활용 예시
- macOS 시스템을 대상으로 보안 점검이 필요할 때
- 정보유출 의심 로그가 발생했을 때
